# Tienda Medica — Usando React con Vite

## Estructura de directorios

```text
med-store/
├─ public/
├─ src/
│  ├─ assets/               # imagenes
│  ├─ components/           # NavBar
│  ├─ pages/                # PaginaUno, PaginaDos, PaginaTres
│  ├─ styles/               # CSS global sencillo
│  ├─ App.jsx
│  └─ main.jsx
├─ index.html               # root donde se renderiza todo.
├─ package.json             # dependencias
├─ vite.config.js           # config de vite
└─ README.md            
```

## Requisitos

- Node.js
- npm

## Instalación

```bash
npm install
npm run dev
```

Ir a `http://localhost:5173` en el navegador!
